# AstroAssist 晴天钟助手 🔭

天文爱好者专属的气象预报插件，基于 **ShinBot** 框架开发。调用 **Open-Meteo** 的 ECMWF 高精度数据与 **7Timer!** 专业天文指标，输出看板式 PNG 预报图。

## ✨ 核心特性

- **专业气象指标**：总云量、低/中/高空云量、气温、露点温度、相对湿度、风速。
- **天文指标**：视宁度 (Seeing) 与透明度 (Transparency)，来自 7Timer! API。
- **结露风险预警**：根据露点温度自动标记风险等级。
- **高德地图集成**：支持地名定位，内置 GCJ-02 → WGS-84 坐标转换。
- **日夜主题切换**：根据目标位置日出日落自动切换浅色/深色看板。
- **超清渲染**：3 倍超采样输出 2100px 物理宽度 PNG 原图（依赖 RenderKit）。
- **雷达回波**：实时获取中央气象台全国雷达回波拼图。

## 🚀 安装

1. 确保已安装 `shinbot_plugin_renderkit`（提供 Playwright HTML 渲染）。
2. 在 ShinBot 配置中启用本插件。
3. 配置高德地图 API Key（Web服务类型）：
   ```toml
   [[plugins]]
   id = "shinbot_plugin_astroassist"
   enabled = true

   [plugins.config]
   amap_key = "your_amap_api_key"
   ```

## 🎮 指令

| 指令 | 别名 | 说明 |
|---|---|---|
| `!晴天钟` | `!astro` | 获取看板预报 |
| `!晴天钟 -d 1` | | 指定天数 (1-7) |
| `!晴天钟 -n` | | 仅夜间时段 |
| `!晴天钟 西藏阿里` | | 临时查询某地 |
| `!设置位置 北京天文馆` | `!setloc` | 设置默认观测位置 |
| `!设置位置 -c 39.9 116.4` | | 手动坐标 (WGS-84) |
| `!雷达` | `!radar` | 获取最新全国雷达回波拼图 |
| `!雷达动图` | `!radargif` | 全国雷达回波动图 (~2小时动画) |
| `!雷达 华北` | | 区域拼图 (华北/华东/华中/华南/西南/西北/东北) |
| `!雷达 北京` | | 单站雷达 (省份名或城市名) |
| `!晴天钟 help` | | 显示帮助信息 |

## 📊 视觉说明

- **气温/风速**：色阶方块，颜色越深越极端。
- **露点风险**：🟢安全 / 🟡中风险 / 🔴高危（易结露）。
- **云量**：日间蓝天底 + 白色填充；夜间深蓝底 + 灰色填充。
